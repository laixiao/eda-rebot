## 2. 审查通过项

### 电源链

```
VIN(U9 端子) → U4 开关 → VBAT → D1 SS56 防反接 → 8V
8V → U11 TMI3255 降压 → 5V
5V → U5 AMS1117-3.3 → 3.3V   5V → U7/U8 XC6206 → CAM_2V8 / CAM_1V2
```

TMI3255 反馈分压(R17 39k / R18 7.5k → 4.96V)、前馈电容、电感、自举电容、输入输出电容均与规格书推荐值一致;AMS1117、XC6206 配置正常。

### 各模块

| 模块 | 结果 |
|---|---|
| ESP32-S3-WROOM-1-N16R8(U1)最小系统 | 通过。EN 复位 RC、BOOT 上拉、strapping 引脚安全、八线 PSRAM 引脚(IO35/36/37)正确未用 |
| 摄像头 FPC-24P(U6) | 通过。24 脚逐一核对,SCCB 上拉、三路电源、PWDN 接地均正确 |
| 麦克风 INMP441(U2) | 通过。I2S 接 IO16/17/18,L/R 下拉选左声道 |
| 功放 MAX98357(U3) | 通过。GAIN=15dB 档;SD 由 PCA9685 控制,上电默认静音 |
| OLED(U10)+ I2C | 通过。IO12/13 + 10k 上拉;地址 0x3C/0x40 无冲突 |
| PCA9685(U16)+ LED 驱动 | 通过。地址 0x40、OE#/EXTCLK 接地正确;两级低边 MOSFET 开关,上电灯灭 |
| 舵机接口 T3–T7 | 通过。线序与 SG90 一致,每路 10k 下拉防抖 |
| 扩展口 T8–T10、安装件 | 通过 |

### 网络专项

- 无单引脚网络、无孤岛;悬空引脚逐一核查均为合理 NC。
- 电源网络规模:GND 70 结点、3.3V 25、5V 23、8V 8,与器件清单吻合。
- 位号无重复(R20/U13 等跳号为纯编号空缺,不影响)。
