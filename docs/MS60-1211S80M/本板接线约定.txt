MS60-1211S80M 60G 雷达 · 临时接线约定
================================

电源（可借舵机座）
  雷达 VCC  →  本板 3V3（模块手册允许 3.0~5.5V）
  雷达 GND  →  舵机座 GND
  舵机座   →  不接（且勿对该路开舵机）

信号
  雷达 GPIO-OUT  →  ENC3_A（XL9555 IO0_0）
  雷达 TX        →  ENC1_A = U1 IO9（ESP32 收）
  雷达 RX        →  ENC1_B = U1 IO10（ESP32 发）

不要接
  T2（RXD0/TXD0）—— 那是下载口，不是雷达口

调试
  首页「60G 雷达」或打开 /radar；固件固定使用已验证的 115200 8N1、正常极性
  固件每 200ms 自动发送一次只读 0x30 检测查询，页面无需 UART 配置
  主页面总开关只暂停/恢复雷达采集，不切断 VCC，UART 和 ENC1 占用不变
  0x59/0x30 单目标回复已验证；TYPE=5 多目标仍待完整验收，slot 不是稳定目标 ID
  占用 ENC1 时，ENC1 编码器不可用
