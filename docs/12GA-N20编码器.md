# 12GA-N20 编码减速电机

N20 直流减速电机，尾部集成磁编码器，6 线引出。配套资料见同目录 `12GA-N20编码器.webp`（接线与参数）、`12GA-N20产品规格.webp`（外形尺寸）。

## 线色定义

| 线色 | 功能 | 说明 |
|------|------|------|
| 红 | 电机电源 + | 与白色对调可改变电机转向 |
| 白 | 电机电源 − | 与红色对调可改变电机转向 |
| 黑 | 编码电源 + | **不可接反**，工作电压 3.3～5 V |
| 蓝 | 编码电源 − | **不可接反**，工作电压 3.3～5 V |
| 黄 | 信号反馈 | AB 相方波之一，电机轴每转 7 个脉冲 |
| 绿 | 信号反馈 | AB 相方波之一，电机轴每转 7 个脉冲 |

## 电气参数

| 项目 | 规格 |
|------|------|
| 电机类型 | N20 直流减速电机 |
| 输出信号 | 方波 AB 相 |
| 编码器供电 | 3.3 V / 5 V |

## 接线要点

1. **电机**：红（+）/ 白（−）接驱动电源；仅交换红、白可反转，勿与编码器线混接。
2. **编码器**：黑（+）/ 蓝（−）接 3.3 V 或 5 V 逻辑电源，**极性必须正确**。
3. **AB 信号**：黄、绿接 MCU 或计数器输入；具体 A/B 相序以实测为准，软件可反相校正。
4. **共地**：编码器地与主控 GND 共地；电机驱动地建议与逻辑地单点汇合，避免大电流地线干扰编码器信号。

## 原理图接法参考

```
电机驱动 OUT+ ── 红
电机驱动 OUT− ── 白

3.3V/5V ── 黑（编码+）
GND     ── 蓝（编码−）

MCU GPIO ── 黄（编码 A 或 B）
MCU GPIO ── 绿（编码 B 或 A）
```

> 本板若经驱动芯片（如 H 桥）控制，电机红/白接驱动输出；编码器黑/蓝接 ESP32 侧 3.3 V 与 GND，黄/绿接可用 GPIO（需支持输入/中断）。
