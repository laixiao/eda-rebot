# SPI 显示模块 + 转接板

3.5 寸 40PIN 屏（HSD 320xRGBx480，**ST7796 + 电阻触摸版**）+ 4.0 寸 40PIN SPI 转接板 V1.3（料号 40002），成套使用。
实物照片：`SPI显示模块转接板.jpg`；屏规格书：`ST7796-带电阻触摸.jpg`。

## 屏规格（2026-07-09 规格书确认）

| 项 | 值 |
|----|----|
| 面板 | HSD 3.5 寸，320xRGBx480 |
| 驱动 IC | **本套为 ST7796**（规格书另有 ILI9488 可选款，固件读 RDDID 兜底） |
| 触摸 | **本套为电阻触摸**（转接板 U1 = XPT2046 型触摸 ADC，SPI 接口，与屏共总线） |
| 接口 | 40PIN：8080 并口 或 SPI（3/4 线），转接板固定为 SPI |
| 背光 | 6 颗 LED，共 90mA |
| 电压 | VCC = 2.8~3.3V（**不可接 5V**） |
| 功耗 | 0.33W |

## J1 排针（14 Pin，实物丝印右 1 左 14）

| Pin | 信号 | 本板应接 | 说明 |
|-----|------|---------|------|
| 1 | VCC | 3.3V | 2.8~3.3V |
| 2 | GND | GND | 地 |
| 3 | CS | LCD_CS | 屏 SPI 片选 |
| 4 | RESET | LCD_RST | 屏复位 |
| 5 | DC | LCD_DC | 数据/命令 |
| 6 | SDI (MOSI) | LCD_MOSI | SPI 数据入 |
| 7 | SCK | LCD_SCK | SPI 时钟 |
| 8 | LED | LCD_LED | 背光使能（板载 Q1 驱动管，逻辑输入） |
| 9 | SDO (MISO) | LCD_MISO | SPI 数据出 |
| 10 | **T_CLK** | LCD_SCK | 触摸共用 SPI 时钟（读触摸时降 ≤2MHz） |
| 11 | **T_CS** | T_CS（XL9555 IO1_0） | 触摸片选 |
| 12 | **T_SDI** | LCD_MOSI | 触摸共用 MOSI |
| 13 | **T_SDO** | LCD_MISO | 触摸共用 MISO（未选中时三态） |
| 14 | **T_IRQ** | T_IRQ（XL9555 IO1_5） | 触摸笔按中断，开漏输出需上拉（转接板一般已带） |

注意 10~14 顺序：**10=T_CLK … 14=T_IRQ**，与部分资料相反，以实物丝印为准。

若换电容触摸款（FT6336U，I2C 0x38），10~14 变为 SCL / RST / SDA / NC / INT，接法完全不同，需重新审查。

## SD 卡（J3，可选）

SD_CS、SD_MOSI、SD_MISO、SD_SCK — 与屏共用 SPI 时软件需分片选。本板未使用。
