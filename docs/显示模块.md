# 4.0 寸 SPI 显示转接板

4.0 寸 40PIN 屏 + SPI 转接板，型号 V1.3（料号 40002）。参考图：`显示模块.jpg`。

## 屏侧

40P FPC 接 LCD；副 FPC 可接触摸排线。

## J1 排针（14 Pin）

| Pin | 信号 | 说明 |
|-----|------|------|
| 1 | VCC | 电源 |
| 2 | GND | 地 |
| 3 | CS | 屏 SPI 片选 |
| 4 | RESET | 复位 |
| 5 | DC | 数据/命令 |
| 6 | SDI (MOSI) | SPI 数据出 |
| 7 | SCK | SPI 时钟 |
| 8 | LED | 背光 |
| 9 | SDO (MISO) | SPI 数据入 |
| 10～14 | T_* | 触摸 SPI / 中断 |

## 触摸引脚映射

| 转接板 | 电阻触摸 | 电容触摸 |
|--------|----------|----------|
| T_IRQ | T_IRQ | INT |
| T_SDO | T_SDO | NC |
| T_SDI | T_SDI | SDA |
| T_CS | T_CS | RST |
| T_CLK | T_CLK | SCL |

## SD 卡（J3，可选）

SD_CS、SD_MOSI、SD_MISO、SD_SCK — 与屏共用 SPI 时软件需分片选。
