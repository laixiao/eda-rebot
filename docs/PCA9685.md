# PCA9685PW（I2C PWM 驱动）

NXP 16 路 12 位 PWM 控制器，用于 LED 调光或舵机控制。参考图：`原装正品 PCA9685PW,118 I2C LED驱动芯片 TSSOP-28封装.png`。

## 规格

| 项目 | 参数 |
|------|------|
| 型号 | PCA9685PW,118 |
| 封装 | TSSOP-28 |
| 通道 | 16 路 PWM |
| 分辨率 | 12 bit |
| 接口 | I2C |
| 逻辑电平 | 3.3 V / 5 V 兼容 |

## 接线

| 引脚 | 接法 |
|------|------|
| VCC | 3.3 V 或 5 V（与逻辑一致） |
| GND | 地 |
| SDA / SCL | I2C 总线 |
| OE | 输出使能（可接 GND 常开） |
| PWM0～15 | LED 阴极或舵机信号 |

默认 I2C 地址 **0x40**（A0～A5 可改址）。同一 I2C 总线勿与其他器件地址冲突。
